# Okos locsoló rendszer

GKNB_INTM020 - Mikroelektromechanikai rendszerek

Ez a szoftver egy Automata rendszer szoba növények öntözéséhez készült.


## Alkatrészek:
- Arduino UNO
- Hőmérséklet szenzor
- Fényérzékelő
- Talajnedvesség mérő
- Kijelző(LED)
- Nyomó gomb
- Szivattyú
- Jumper kábelek

## Szoftver:
- Arduino IDE
